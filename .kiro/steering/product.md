# Product Overview

EnvindoSystem is a full-stack web application for environmental monitoring and management. The system provides user authentication, data management, and administrative capabilities for Enviro Metro Lestari.

## Key Features

- User authentication with JWT tokens
- Email verification system
- Role-based access control (admin/user)
- Environmental data tracking and management
- Responsive web interface

## Deployment

- Production URL: https://dev.envirometrolestari.com
- Backend serves API at `/api` endpoint
- Frontend served from `/frontend` path
- Uses Apache with .htaccess for routing
