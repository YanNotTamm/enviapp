Error: "No routes matched location"
-----------------------------------
Penyebab: Frontend belum sync ke server

Solusi:
1. Tunggu 3-5 menit untuk sync
2. Clear browser cache (Ctrl+Shift+Delete)
3. Hard refresh (Ctrl+Shift+R)
4. Buka incognito mode
5. Test lagi

Error: "Invalid verification token"
-----------------------------------
Penyebab: Token sudah digunakan atau salah

Solusi:
1. Register user baru dengan email berbeda
2. Gunakan verification link yang baru
3. Jangan klik link lebih dari 1x

Error: Email tidak masuk
------------------------
Penyebab: Email masuk ke spam atau delay

Solusi:
1. Cek folder Spam/Junk
2. Tunggu 1-2 menit
3. Cek email address benar
4. Test send email: /api/test/send-email?to=your-email

Error: Cannot login after verify
---------------------------------
Penyebab: Email belum ter-verify atau password salah

Solusi:
1. Pastikan klik verification link
2. Pastikan muncul "Verification Successful"
3. Cek password benar (case-sensitive)
4. Try forgot password jika lupa
